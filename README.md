# distributed_spider #

### 1.启动步骤
1.1 pip install -r requirements.txt <br>
1.2 修改config中数据库地址及用户名密码等<br>
1.3 若使用代理，请先启动代理服务器<br>
1.4 python main_scheduler_module.py
